| --------------- | -------------- | ------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ | ----------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Buat Tiket Baru | User (Pelapor) | Use case ini menggambarkan cara user (pelapor) membuat laporan insiden siber baru melalui portal. | User (pelapor) sudah login dan berada di tampilan Portal atau Dashboard. | User (pelapor) berhasil membuat tiket baru dan sistem mengirimkan notifikasi. | 1.
