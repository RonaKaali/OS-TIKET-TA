# Logo Kalselprov

Silakan simpan gambar logo/lambang Provinsi Kalimantan Selatan di folder ini dengan nama:

**`logo-kalselprov.png`**

Format yang didukung:

-   PNG (direkomendasikan)
-   JPG/JPEG
-   SVG

Jika menggunakan format selain PNG, ubah nama file di semua view yang menggunakan logo:

-   `resources/views/welcome.blade.php`
-   `resources/views/layouts/agent.blade.php`
-   `resources/views/components/agent-layout.blade.php`
-   `resources/views/layouts/admin.blade.php`
-   `resources/views/components/admin-layout.blade.php`
-   `resources/views/layouts/guest.blade.php`

atau ubah ekstensi file di semua tag `<img src="{{ asset('images/logo-kalselprov.png') }}">` sesuai format file yang digunakan.
